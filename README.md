## HTTP Explained

Interactive single-page guide that breaks down the Hypertext Transfer Protocol for modern web developers. Built with Next.js 16, the app walks through lifecycle stages, method semantics, status codes, headers, caching strategies, and security best practices.

### Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to explore the guide.

### Production build

```bash
npm run build
npm start
```

### Deploy

The project is optimized for Vercel. After building locally, deploy with:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-205268ca
```
